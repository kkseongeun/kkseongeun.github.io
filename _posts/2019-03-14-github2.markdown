이제는 손에 익었지만 혹시 모르니까 기록하는 글이다. 깃 명령어를 이용해 웹에 `push`하는 방법!

기본적인 깃 명령어
----------------

### First commit
$ git remote add origin https://kkseongeun.github.io  
$ git add .  
$ git commit -m "Initialize Blog"  
$ git push  


### modified
$ git status  
$ git add .  
$ git status  
$ git commit -m " "  
$ git push
